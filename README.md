## Example - 1
```
inicio
  inteiro:a;
  imprima (“digite um valor para a:”);
  leia(a);
  se a = 5
  então
    escreva (“igual a 5”);
  senão
    escreva (“diferente de 5”);
  fim_se
fim
```
